# Claude Code Plugins

A collection of Claude Code plugins by [@adeotek](https://github.com/adeotek).

## Install this marketplace

```
/plugin marketplace add adeotek/claude-code
```

## Plugins

### [context-checkpoint](plugins/context-checkpoint/)

Save and restore named session checkpoints so you can pick up exactly where you left off across Claude Code sessions.

```
/plugin install context-checkpoint@adeotek-plugins
```
